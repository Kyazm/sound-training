# ジャズハーモニー・音感トレーニング詳細調査

## 1. コード進行パターン（難易度別）

### Beginner（初級）

#### 12小節ブルース（ベーシック）
```
| I7  | IV7 | I7  | I7  |
| IV7 | IV7 | I7  | I7  |
| V7  | IV7 | I7  | V7  |
```

#### ジャズブルース
```
| I7    | IV7   | I7    | I7    |
| IV7   | #IVdim7| I7   | vi7 II7|
| ii7   | V7    | I7 VI7| ii7 V7 |
```

#### バードブルース / Blues for Alice (Charlie Parker, Key: F)
```
| FMaj7    | Em7b5 A7  | Dm7 G7   | Cm7 F7   |
| Bb7      | Bbm7 Eb7  | Am7 D7   | Abm7 Db7 |
| Gm7      | C7        | FMaj7    | Gm7 C7   |
```
特徴: ii-V チェーンによる半音下行のクロマティックベースライン

#### 基本ii-V-Iパターン
```
Major: Dm7 - G7 - CMaj7
Minor: Dm7(b5) - G7(b9) - Cm(Maj7)
```

#### I-vi-ii-V ターンアラウンド
```
CMaj7 - Am7 - Dm7 - G7
```

---

### Intermediate（中級）

#### Rhythm Changes (I Got Rhythm, Bb)
```
A Section:
| BbMaj7 G7 | Cm7 F7 | BbMaj7 G7 | Cm7 F7 |
| Fm7 Bb7   | EbMaj7 Ab7| BbMaj7 G7 | Cm7 F7 |

B Section (Bridge):
| D7  | D7  | G7  | G7  |
| C7  | C7  | F7  | F7  |
```

#### トライトーンサブスティテューション
```
基本: ii7 - V7 - IMaj7
トライトーンサブ: ii7 - bII7 - IMaj7
例: Dm7 - Db7 - CMaj7
（G7 の代わりに Db7。共有するトライトーン: B-F）

応用:
Dm7 - Db7 - CMaj7        （V7のサブ）
Abm7 - Db7 - CMaj7       （ii-V 全体のサブ）
Dm7 - G7 - GbMaj7        （Iのサブ…モーダル的）
```

#### セカンダリードミナント
```
I - III7 - vi - II7 - V - I
CMaj7 - E7 - Am7 - D7 - G7 - CMaj7

I - VI7 - II7 - V7 - I
CMaj7 - A7 - D7 - G7 - CMaj7 （ドミナントのチェーン）
```

#### モーダルインターチェンジ（借用和音）
```
Major key で minor key から和音を借りる:

I - bVII - IV - I          CMaj7 - Bb7 - FMaj7 - CMaj7
I - iv - I                  CMaj7 - Fm7 - CMaj7
I - bVI - bVII - I          CMaj7 - AbMaj7 - BbMaj7 - CMaj7
I - bIII - IV - I           CMaj7 - EbMaj7 - FMaj7 - CMaj7
```

#### バックドアii-V
```
iv7 - bVII7 - IMaj7
Fm7 - Bb7 - CMaj7
（マイナーから Major Iへの解決。ソフトで浮遊的な解決感）
```

#### ディセプティブ（偽終止）解決パターン
```
ii - V - vi      (Dm7 - G7 - Am7)
ii - V - bVI     (Dm7 - G7 - AbMaj7)
ii - V - IV      (Dm7 - G7 - FMaj7)
ii - V - #IV     (Dm7 - G7 - F#m7)   ← コルトレーン的
```

---

### Advanced（上級）

#### コルトレーンチェンジ（Giant Steps）
長3度サイクルによる転調:
```
| BMaj7 D7 | GMaj7 Bb7 | EbMaj7 | Am7 D7 |
| GMaj7 Bb7 | EbMaj7 F#7 | BMaj7 | Fm7 Bb7 |
| EbMaj7 | Am7 D7 | GMaj7 | C#m7 F#7 |
| BMaj7 | Fm7 Bb7 | EbMaj7 | C#m7 F#7 |
```

原理: B → G → Eb → B（長3度ずつ下行/上行）

#### クロマティックメディアント
```
I - bIII - I     CMaj7 - EbMaj7 - CMaj7    (半音3つ上)
I - III - I      CMaj7 - EMaj7 - CMaj7     (長3度上)
I - bVI - I      CMaj7 - AbMaj7 - CMaj7    (短6度上)
I - VI - I       CMaj7 - AMaj7 - CMaj7     (長6度上)
```

#### ディミニッシュサブスティテューション
```
dim7コードの対称性を利用:
C#dim7 = E dim7 = G dim7 = Bb dim7
→ C7, Eb7, F#7, A7 いずれにも解決可能

パッシングディミニッシュ:
I - #Idim7 - ii    CMaj7 - C#dim7 - Dm7
IV - #IVdim7 - I/5  FMaj7 - F#dim7 - C/G
```

#### マルチトニックシステム
```
3つのキーセンター間を移動:
|: CMaj7 | Am7 D7 | GMaj7 | Em7 A7 | DMaj7 | Bm7 E7 :|
（C → G → D の連鎖的 ii-V-I）
```

## 2. ジャズスタンダード進行分析

### Autumn Leaves (Key: G minor / Bb Major)
```
| Cm7  | F7    | BbMaj7 | EbMaj7 |
| Am7b5| D7    | Gm     | Gm     |
| Cm7  | F7    | BbMaj7 | EbMaj7 |
| Am7b5| D7    | Gm     | Gm     |
```
**学習ポイント**: Major ii-V-I と minor ii-V-i の交互パターン

### All The Things You Are (Key: Ab, 36小節 AABA)
```
A1: | Fm7    | Bbm7  | Eb7    | AbMaj7 |    ← vi-ii-V-I in Ab
    | DbMaj7 | G7    | CMaj7  | CMaj7  |    ← IV, then V-I in C

A2: | Cm7    | Fm7   | Bb7    | EbMaj7 |    ← vi-ii-V-I in Eb
    | AbMaj7 | D7    | GMaj7  | GMaj7  |    ← IV, then V-I in G

B:  | Am7    | D7    | GMaj7  | GMaj7  |    ← ii-V-I in G
    | F#m7b5 | B7    | EMaj7  | C7+    |    ← ii-V-I in E, V+ back

A3: | Fm7    | Bbm7  | Eb7    | AbMaj7 |    ← vi-ii-V-I in Ab
    | DbMaj7 | Gb7   | Cm7    | Bdim7  |
    | Bbm7   | Eb7   | AbMaj7 | Gm7b5 C7|   ← turnaround
```
**学習ポイント**: Ab→C→Eb→G→E→Ab と複数キーにまたがる ii-V-I チェーン

### Blue Bossa (Key: C minor)
```
| Cm7  | Cm7   | Fm7   | Fm7   |
| Dm7b5| G7    | Cm7   | Cm7   |
| Ebm7 | Ab7   | DbMaj7| DbMaj7|
| Dm7b5| G7    | Cm7   | Dm7b5 G7|
```
**学習ポイント**: マイナーキーのii-V-i + 半音上への転調

### Stella By Starlight (Key: Bb)
```
| Em7b5 | A7b9  | Cm7   | F7    |
| Fm7   | Bb7   | EbMaj7| Ab7#11|
| BbMaj7| Em7b5 A7| Dm7  | G7    |
| Cm7   | F7    | BbMaj7| BbMaj7|
```
**学習ポイント**: セカンダリードミナント多用、#11テンション

## 3. コードボイシング認識

### 聴き取りで重要なボイシングの違い

#### ルートポジション vs 転回形
- **Root Position**: 安定、重い、明確
- **1st Inversion (第1転回)**: 3rdがベース → 軽い、前進感
- **2nd Inversion (第2転回)**: 5thがベース → 浮遊感
- **3rd Inversion (第3転回)**: 7thがベース → 強い前進感（次のコードへ解決）

#### Shell Voicing（シェルボイシング）
3rdと7thだけを残した最小限のボイシング:
```
Dm7:  F + C  (3rd + b7)
G7:   B + F  (3rd + b7)
CMaj7: E + B  (3rd + 7th)
```
**ガイドトーンの半音進行**: F→E (3rd下行), C→B (7th下行) in ii-V-I

#### Drop 2 Voicing
クローズボイシングの上から2番目の音をオクターブ下に:
```
CMaj7 close (top→bottom): B-G-E-C
2nd from top (G) をオクターブ下に drop:
Drop 2 (bottom→top): G-C-E-B
```
ジャズギター・ピアノの基本ボイシング。広がりのある響きになる

## 4. ファンクショナルハーモニー聴き取り

### コード機能の3カテゴリ

| 機能 | コード | 感覚 |
|------|--------|------|
| **Tonic (安定)** | I, iii, vi | 到着感、安心 |
| **Subdominant (移行)** | ii, IV | 動き出し、開放 |
| **Dominant (緊張)** | V, vii° | 緊張、解決要求 |

### ベースラインパターンの認識

| パターン | 例 | 特徴 |
|---------|-----|------|
| 下行5度 | C→F→Bb→Eb | サークルオブ5th、最も自然 |
| 半音下行 | C→B→Bb→A | クロマティック、スムーズ |
| ペダルポイント | C→C→C→C | ベース持続、上で和声変化 |
| ステップワイズ | C→D→E→F | スケール的ベース |
| 3度進行 | C→E→G→B | メディアント関係 |

## 5. ネオソウル / コンテンポラリー和声

### ネオソウル特有のサウンド

```
Major9:          CMaj9 (ネオソウルのトニック定番)
Minor9:          Dm9  (ドリアン的浮遊感)
dom9(#11):       G9(#11) (リディアンドミナント)
min7(11):        Am11 (クォータルハーモニー風)
Maj7(#5):        CMaj7#5 (リディアンオーギュメント)

ネオソウル定番進行:
IVMaj7 - iii7 - vi7 - V7
IIm9 - V13 - IMaj9 - (VIm9)
I - bII - I (半音スライド)
```

### エリカ・バドゥ / ディアンジェロ的ハーモニー
```
ミクソリディアン的: I7 - IV7 系 (ブルース由来)
マイナーペンタ + テンション: min11, min9(maj7)
クォータルハーモニー: 4度堆積 (例: D-G-C-F)
```

## 6. リズムトレーニング

### ジャズリズムの基本要素

| 要素 | 説明 |
|------|------|
| **スウィング** | 3連符ベースの不均等な8分音符 |
| **シンコペーション** | 裏拍アクセント |
| **アンティシペーション** | コードチェンジの先取り |
| **ポリリズム** | 3対4, 4対3等 |
| **メトリックモジュレーション** | テンポの知覚的変化 |
| **クラーベ** | アフロキューバン (2-3, 3-2) |

### リズム聴き取り難易度

1. **初級**: 4/4拍子、4分・8分音符主体
2. **中級**: シンコペーション、3連符、付点リズム
3. **上級**: ポリリズム、変拍子(5/4, 7/8)、スウィング度合い
4. **マスター**: メトリックモジュレーション、フリータイム認識
