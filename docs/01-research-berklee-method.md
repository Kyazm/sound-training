### バークリーの教育哲学

- **Functional Hearing（機能的聴音）**: インターバルを孤立して覚えるのではなく、**調性の文脈の中で**聴く
- **Movable Do（移動ド）ソルフェージュ**: バークリーは移動ドシステムを採用。クロマティックシラブル付き
- **Inner Hearing**: 音を鳴らす前に頭の中で鳴らす訓練を重視
- **Vertical + Horizontal Approach**: コードを「スナップショット」として垂直的に聴く + ボイスリーディングを水平的に追跡する
- **Real Music Context**: 練習曲ではなく、実際のジャズスタンダードや楽曲から素材を取る

### 機能的聴音（Functional） vs 音程的聴音（Intervallic）

| 側面 | Intervallic | Functional（バークリー式） |
|------|-------------|--------------------------|
| 基準点 | 直前の音からの距離 | キーセンター（トニック） |
| 認識方法 | 「これは短3度」 | 「これはキーの3度（Mi）」 |
| 強み | 無調音楽に対応可能 | 実際の音楽でのコード/メロディ認識に直結 |
| 弱点 | 文脈なしでは実践的でない | 無調音楽には適用しにくい |
| 適用 | 現代音楽、無調 | ジャズ、ポップス、クラシック調性音楽全般 |

### ジャズ特有のインターバルトレーニング

バークリーでは、単独インターバルだけでなく**コードトーンとの関係**でインターバルを聴く訓練をする：

- **b9 (♭9)**: ドミナント7thの上で → 強い緊張感、アルタードサウンド
- **#9 (♯9)**: ドミナント7thの上で → ジミヘンコード、ブルージーなサウンド
- **#11 (♯11)**: リディアン・ドミナントの特徴音 → 浮遊感
- **b13 (♭13)**: アルタードドミナントの特徴 → ダークな解決感
- **コンパウンドインターバル**: 9th(=M2+octave), 11th(=P4+octave), 13th(=M6+octave)

### 機能的インターバルトレーニング（Functional Approach）

従来の方法：2音を聞いて「何度か」を答える
バークリー式：**キーの中で**各音のスケールディグリーを認識し、結果としてインターバルがわかる

例: Key=Cで「E→B」が鳴る → 「mi→ti」= P5 と認識

| diminished | R-m3-dim5 | 不安定、緊張 |

| コードタイプ | 構成 | ダイアトニック機能 | サウンドの特徴 |
|-------------|------|-----------------|--------------|
| Maj7 | R-M3-P5-M7 | Iコード, IVコード | 温かい、洗練された |

| min7 | R-m3-P5-m7 | iiコード, iiiコード, viコード | ソフト、メランコリック |

## 4. コード進行トレーニング

### レベル1: ダイアトニック基本進行

```
I - IV - V - I          （基本カデンツ）
I - vi - IV - V          （50sプログレッション）
I - V - vi - IV          （ポップ定番）
ii - V - I               （ジャズ最重要）
I - vi - ii - V          （ターンアラウンド）
vi - IV - I - V          （現代ポップ）
```

### レベル2: セカンダリードミナント

```
I - VI7 - ii - V - I     （VI7 = セカンダリーdom of ii）
I - III7 - vi - ii - V - I
I - II7 - V - I          （II7 = V/V）
I - IV - #IVdim7 - I/5   （パッシングディミニッシュ）
```

### レベル3: ジャズ基本進行

```
ii-V-I (メジャー):        Dmin7 - G7 - Cmaj7
ii-V-i (マイナー):        Dmin7(b5) - G7(b9) - Cmin(maj7)
iii-vi-ii-V-I:           Emin7 - Amin7 - Dmin7 - G7 - Cmaj7
I-VI7-II7-V7 ターンアラウンド
12小節ブルース（ジャズ版）
Rhythm Changes (I Got Rhythm)
```

### レベル4: 高度なジャズ進行

```
トライトーンサブ:         ii - bII7 - I （例: Dmin7 - Db7 - Cmaj7）
バックドアii-V:          bVIImin7 - bVII7 - I （例: Bbmin7 - Bb7 - Cmaj7 → 実質 iv-bVII7-I）
ディセプティブ解決:       ii - V - vi  or  ii - V - bVI
コルトレーンチェンジ:     BMaj7 - D7 - GMaj7 - Bb7 - EbMaj7 - F#7 - BMaj7（長3度サイクル）
クロマティックメディアント: I - bIII - I  or  I - #V - I
```

### レベル5: スタンダード進行パターン

**Autumn Leaves** (メジャーとマイナーのii-V交互):
```
ii-V-I (Bb) → ii-V-i (Gm) のサイクル
Cmin7 - F7 - BbMaj7 - EbMaj7 - Amin7(b5) - D7 - Gmin
```

**All The Things You Are** (複数キーのii-V-I):
```
Fmin7 - Bbmin7 - Eb7 - AbMaj7 - DbMaj7 - G7 - CMaj7 ...
（Ab → C → Gb → E → Ab と転調）
```

**Giant Steps** (コルトレーンチェンジ):
```
BMaj7 - D7 - GMaj7 - Bb7 - EbMaj7 - Am7 - D7 - GMaj7 ...
```

## 6. モードトレーニング

### 7つのチャーチモード

| モード | 特徴音（Characteristic Note） | サウンドの特徴 | 代表曲 |
|--------|------------------------------|--------------|--------|
| Ionian | M7 (Ti) | 明るい、スタンダード | Happy Birthday |
| Dorian | M6 + m3 (La + Me) | マイルド・マイナー、ファンキー | So What |
| Phrygian | m2 (Ra) | スパニッシュ、エキゾチック | White Rabbit |
| Lydian | #4 (Fi) | 浮遊、夢幻的 | Simpsons Theme |
| Mixolydian | m7 (Te) | ブルージー、ロック的 | Norwegian Wood |
| Aeolian | m6 + m3 (Le + Me) | ナチュラルマイナー | Stairway to Heaven |
| Locrian | m2 + dim5 (Ra + Se) | 最も不安定、理論的 | 実用曲少ない |

### メロディックマイナーモード

| モード | 元スケール | 特徴 | 使用場面 |
|--------|-----------|------|---------|
| Melodic Minor | 1st mode | マイナーだがM6, M7 | min(Maj7)コード上 |
| Dorian b2 | 2nd mode | ドリアン+b2 | sus(b9)コード上 |
| Lydian Augmented | 3rd mode | リディアン+#5 | Maj7(#5)コード上 |
| **Lydian Dominant** | 4th mode | リディアン+b7 | dom7(#11)コード上 |
| Mixolydian b6 | 5th mode | ミクソ+b6 | dom7(b13)コード上 |
| Locrian ♮2 | 6th mode | ロクリアン+M2 | min7(b5)コード上 |
| **Altered Scale** | 7th mode | 全てのテンションがアルタード | dom7alt コード上 |

### その他の重要スケール

- **ディミニッシュスケール**: 半全（H-W）と全半（W-H）の2種
- **ホールトーン**: dom7(#5)上で使用
- **ブルーススケール**: 1-b3-4-#4-5-b7
- **ビバップスケール**: メジャー/ドミナント/マイナー各バリエーション

## 7. 既存アプリの分析

| アプリ | 長所 | 短所 |
|--------|------|------|
| **EarMaster** | 2500+エクササイズ、MIDI対応、適応型AI、学校採用実績多数 | UIが複雑、有料(月$3.99) |
| **ToneGym** | ブラウザベース、ゲーミフィケーション、コミュニティ機能 | 高価(月$13.95)、ゲーム的すぎる |
| **tonedear** | 無料Web、ファンクショナル音程対応、メロディックディクテーション | 体系的カリキュラムなし、UIが素朴 |
| **Functional Ear Trainer** | ファンクショナルアプローチに純粋特化 | 音程認識のみ、コード進行なし |
| **iReal Pro** | ジャズ進行練習に最適 | 耳のトレーニングではない |
| **Perfect Ear** | iOS無料、大規模カリキュラム | 技術的不具合報告あり |
| **Berklee Pulse** | バークリー公式無料ツール(pulse.berklee.edu) | 機能が限定的 |

### 既存アプリに不足している要素（本アプリのチャンス）

1. **ジャズ特化の体系的コード進行トレーニング**がほぼ皆無
2. **テンション付きコード**の聴き取り練習ができるアプリがない
3. **モーダルインターチェンジ**の聴き取り練習がない
4. **ネオソウル / コンテンポラリーR&B**的な和声トレーニングがない
5. **アダプティブ難易度**+**間隔反復学習**の組み合わせが弱い
6. **コンテキスト付きインターバル**（キーの中でのインターバル認識）が少ない
