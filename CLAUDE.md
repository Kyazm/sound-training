# 音感トレーニングアプリ
バークリーメソッド含めjazzや最新のネオソウルのセッションに対応できるような音感を磨くためのwebアプリケーション。
ダイアトニック以外のインターバルのトレーニング、コード進行のトレーニングをメインとする。

## 技術スタック
- Vite + React 19 + TypeScript + Tailwind CSS 4
- React Router 7 (HashRouter)
- Zustand (状態管理)
- Supabase (Auth + PostgreSQL)
- Tone.js (オーディオ)
- Vitest + React Testing Library
- Biome (linter/formatter)
- pnpm
- GitHub Pages デプロイ

## UI
- 日本語
- 音楽用語は英語表記（Maj7, ii-V-I 等）
